📄 ZARUverse Whitepaper

Abstract

ZARUverse یک اکوسیستم برند-محور روی BNB Chain است که با دو توکن ZARU و DMD و نقش Relic (Arxium) اداره می‌شود. ZARU نبض تعاملات است؛ DMD تاج حاکمیت. رأی‌دهی در Snapshot و اجرا از طریق قراردادها و سیاست‌های خزانه صورت می‌گیرد.

Network

BNB Smart Chain (Chain ID: 56). سازگار با EVM، کارمزد پایین، مناسب برای DeFi/NFT/DAO.

BSCScan Snapshot: zaruverse.bnb

Tokens

ZARU: توکن Utility برای تعاملات، رأی عمومی، staking و lore unlock.

DMD: توکن Prestige برای رأی طلایی، کنترل خزانه، دسترسی‌های ویژه.

Governance

  • Model: Dual-token + Relic gating
  • Voting: Snapshot (zaruverse.bnb)
  • Weights: ZARU = رأی عمومی، DMD = رأی طلایی
  • Gating: Arxium برای فعال‌سازی رأی (در صورت تعریف)
Governance page Vote on Snapshot

Relics

Relicها کلیدهای حاکمیتی‌اند. Arxium رأی را فعال می‌کند؛ Vaultbound حافظه را می‌گشاید. نسل‌های بعدی Relic، دسترسی‌ها و آیین‌های تازه می‌آورند.

Treasury

  • Sources: کارمزدها، فروش NFT/توکن، مشارکت‌ها
  • Uses: توسعه پروتکل، هنری/برند، نقدینگی، عملیات
  • Controls: Proposal → Snapshot → Execution policy

Roadmap

  • Phase I: راه‌اندازی وب + توکن‌ها + Snapshot
  • Phase II: Relic utilities + staking + lore unlock
  • Phase III: DAO عملیات خزانه + ادغام‌های صرافی/NFT

Security & audit

  • Best practices: حداقل دسترسی، timelock، چندامضایی برای خزانه
  • Audit: افزودن گزارش‌های ممیزی پس از انتشار رسمی
